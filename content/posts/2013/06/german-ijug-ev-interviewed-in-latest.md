 The Interest Alliance of the Java User Group (iJUG) is an association of twelve Java User Groups in Germany, Austria and Switzerland. &nbsp;The ambition of the iJUG is to represent the interests of more than 20,000 Java User and the Java User Groups. If you want to know more about them visit the German website <a href="" target="_blank">http://www.ijug.eu</a>
<br>
<br>
 And if you are interested in writing: Take a look at their mouthpiece called "<a href="http://www.ijug.eu/java-aktuell/das-magazin.html" target="_blank">Java Aktuell</a>" which is a German speaking magazine covering all the latest and greatest in the Java ecosystem.
