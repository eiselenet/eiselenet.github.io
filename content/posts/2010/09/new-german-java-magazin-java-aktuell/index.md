It's here! The new magazin (Java Aktuell) from the german iJUG (Interessenverbund der Java User Groups e.V.). 
<div class="separator" style="clear: both; text-align: center;">
 <a href="http://www.ijug.eu/images/vorlagen/2010_java_aktuell_umschlag_web.jpg" imageanchor="1" style="clear: right; float: right; margin-bottom: 1em; margin-left: 1em;"><img border="0" height="200" src="http://www.ijug.eu/images/vorlagen/2010_java_aktuell_umschlag_web.jpg" width="141"></a>
</div>The iJUG was foundet right after the Sun/Oracle merger. Some of the most powerfull German Java User Groups (JUG) and the German Oracle Users Group (DOAG) decided to establish the new association of the Java User Groups e.V. (iJUG). The new magazin is the quarterly voice of the iJUG. More (german) information can be found on the <a href="http://www.ijug.eu">iJUG website</a>.
