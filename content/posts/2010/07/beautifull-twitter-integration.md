This is, what the new TwittFont widget looks like. The (german) <a href="" target="_blank">homepage </a>of the new copperation between a german font provider (<a href="" target="_blank">FontShop</a>) and an agency (<a href="" target="_blank">Moccu</a>) provides a very simple way of configuring your own.
<br>
 Even if you can't read german. This is basically a four step process.
<br>
 1) Enter your search word or username
<br>
 2) select display font
<br>
 3) review the animation
<br>
 4) select size and copy the html embed code
