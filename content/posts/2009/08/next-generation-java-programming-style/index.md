<a href="" target="_blank">Stephan Schmidt</a> wrote a nice blogpost about the<a href="" target="_blank"> next generation Java programming style</a>. Many developers move away from Java to new languages like Ruby, Python, Groovy, Erlang, Scala. If you are trapped with Java, you can still use some of the benefits of those new languages.
<br>
<br>
<ol>
 <br>
 <li>Final is your new love</li>
 <br>
 <li>No setters</li>
 <br>
 <li>Do not use loops for list operations</li>
 <br>
 <li>Use one liners</li>
 <br>
 <li>Use many, many objects with many interfaces</li>
 <br>
 <li>Use Erlang-Style Concurrency</li>
 <br>
 <li>Use Fluent Interfaces</li>
 <br>
 <li>Data Transfer Objects without setters and getters</li>
 <br>
</ol>
