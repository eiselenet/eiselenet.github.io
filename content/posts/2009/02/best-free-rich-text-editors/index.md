Found <a href="http://woork.blogspot.com/2009/02/best-rich-text-editors-ready-to-use-in.html" target="_blank">this post from Antonio Lupetti</a>. He lists the rest Rich Text Editors ready to use in web projects with some comments and hints how to implement them on pages using a few lines of HTML code.
<br>
It seems too easy to use one of them within your software product. But, be aware of the license.
