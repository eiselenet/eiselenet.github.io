Starting in december 2008 <a href="" target="_blank">Joseph D. Darcy</a> announced <a href="" target="_blank">Project Coin</a>. The goal of Project Coin is to determine what set of small language changes should be added to JDK 7. A call for proposals period will run until March 30, 2009. 
<br>
The first 17 proposals are <a href="http://blogs.sun.com/darcy/entry/project_coin_week_1_update" target="_blank">now available at Darcy's blog</a>.
